4. Menu Filtre :

Vous offre la possibilité d’appliquer les filtres de couleurs (gris, négatif, sépia), les filtres de détection de contours (Laplacien, Rehausseur, Robert, Sobel), ainsi de changer la luminosité et le contraste de l’image.
